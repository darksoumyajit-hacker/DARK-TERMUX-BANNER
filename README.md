## 🎯 About

**DARK-SOUMYA** is a professional Termux banner installer that enhances your terminal experience with customizable banners, themes, and ASCII art. Built with Bash scripting, it offers a smooth installation process with multiple styling options.

## ✨ Features

- 🚀 **Custom ASCII Banners** - Multiple banner styles to choose from
- 🎨 **Color Themes** - 5 different themes (Default, Cyber, Sunset, Ice, Matrix)
- 💫 **Loading Animations** - Interactive progress bars and animations
- 🔄 **Auto Backup** - Automatic backup of your existing `.bashrc`
- 📦 **Easy Installation** - One command installation process
- 🗑️ **Clean Uninstall** - Complete removal with restore functionality
- 🎯 **Preview System** - Preview banners before installation
- 📱 **Termux Optimized** - Perfectly tailored for Termux environment

## 📋 Prerequisites

- Termux (Android)
- Git
- Bash 4.0+

## 🚀 Installation

Open Termux and run these commands:

```bash
pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/soumyajit/DARK-SOUMYA.git
cd DARK-SOUMYA
chmod +x install.sh
bash install.sh
